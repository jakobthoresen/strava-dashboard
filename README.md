# Løpedashbord

Personlig treningsdashbord som visualiserer løpedata med interaktive grafer, GPS-kartvisning og detaljert øktanalyse.

## Demo

[strava-dashboard-kappa.vercel.app](https://strava-dashboard-kappa.vercel.app)

## Features

- Aktivitetsoversikt med ukentlig distanse og pulssoner
- GPS-kartvisning av løyper
- Detaljert øktanalyse med splits per kilometer
- Dark/light mode

## Tech stack

- Next.js 14 + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts (grafer)
- Leaflet / react-leaflet (kart)
- Vercel (deployment)

## Kommende features

- Strava OAuth-integrasjon
- Supabase database
